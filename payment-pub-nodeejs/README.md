# payment-pub-nodeejs
Payment System bulit using  Ejs, Express, Nodejs, MongoDBAtlas.
# install your dependencies
npm install
# Check js files to enter your credentials "YOUR RAZORPAY_KEY" "YOUR RAZORPAY_SECRETKEY" "YOUR MONGODB_URI"
# serve app
npm start
